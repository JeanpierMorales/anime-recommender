#🎌 Anime Recommender System
Aplicación web desarrollada con Django que permite recomendar animes personalizados a los usuarios en función de sus gustos, preferencias y comportamiento dentro de la plataforma.
🚀 Descripción
Este proyecto tiene como objetivo implementar un sistema de recomendación de animes, aplicando lógica de filtrado y potencial integración con modelos de inteligencia artificial. Está diseñado como una práctica profesional para fortalecer habilidades en desarrollo web, estructuras de datos y sistemas inteligentes.
⚙️ Tecnologías utilizadas
Python
Django
HTML, CSS
SQLite
Git y GitHub
🧠 Funcionalidades principales
Registro e inicio de sesión de usuarios
Visualización de catálogo de animes
Sistema de recomendaciones personalizadas
Búsqueda y filtrado por géneros
Gestión de favoritos
📦 Instalación
Clonar el repositorio:
git clone https://github.com/tu-usuario/tu-repositorio.git
​
Entrar al proyecto:
cd tu-repositorio
​
Crear entorno virtual:
python3 -m venv env
​
Activar entorno virtual:
source env/bin/activate
​
Instalar dependencias:
pip install -r requirements.txt
​
Ejecutar migraciones:
python manage.py migrate
​
Iniciar servidor:
python manage.py runserver
​
